## COVID-19 Mortality Analysis (R Project)
## Project Overview

Analyzed COVID-19 patient data to identify key risk factors for mortality using R, including age, gender, and country.

**Objectives**

Clean and preprocess data
Calculate mortality rates
Compare mortality by age and gender
Visualize trends over time and by demographics

**Dataset**
Patient-level COVID-19 line-list dataset with variables: age, gender, country, death, reporting.date.
